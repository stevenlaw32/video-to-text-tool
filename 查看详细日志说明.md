# 📖 如何查看 Whisper 详细转录日志

## 🎯 问题说明

在处理视频时，Whisper 会生成详细的逐句转录日志，格式如下：
```
[00:00.000 --> 00:03.000] 大家好，欢迎来到今天的课程
[00:03.000 --> 00:06.500] 今天我们要学习的内容是关于
[00:06.500 --> 00:10.000] Python 编程的基础知识
```

这些详细日志默认只显示在**运行服务器的终端窗口**中。

---

## ✅ 解决方案

### 方法 1：使用专用启动脚本（推荐）

**步骤：**

1. **双击启动脚本**
   - 在 Finder 中找到项目文件夹
   - 双击 `启动服务（带日志窗口）.command`

2. **观察效果**
   - 会自动打开一个**新的终端窗口**
   - 该窗口会显示服务器运行日志
   - 浏览器会自动打开网页界面

3. **查看详细日志**
   - 在网页上传视频并开始处理
   - 切换到新打开的终端窗口
   - 即可看到 Whisper 的实时转录进度

**优点：**
- ✅ 自动化，一键启动
- ✅ 日志窗口独立，不会被其他输出干扰
- ✅ 可以并排查看网页和日志

---

### 方法 2：在 IDE 终端中查看

**步骤：**

1. **在 IDE 中打开终端**
   - VS Code/Cursor: 按 `` Ctrl + ` `` 或点击菜单 `Terminal` → `New Terminal`

2. **启动服务器**
   ```bash
   cd /Users/apple/开发/DevOps/video-to-text-tool
   source venv/bin/activate
   python3 app_advanced.py
   ```

3. **查看日志**
   - 在网页上传视频并开始处理
   - 回到 IDE 的终端窗口
   - 即可看到 Whisper 的实时转录进度

**优点：**
- ✅ 在 IDE 内部，无需切换窗口
- ✅ 可以使用 IDE 的终端功能（搜索、复制等）

---

### 方法 3：使用现有的启动脚本

**步骤：**

1. **打开终端**
   - 按 `Cmd + Space` 搜索 `Terminal` 并打开

2. **运行启动脚本**
   ```bash
   cd /Users/apple/开发/DevOps/video-to-text-tool
   ./启动服务.sh
   ```

3. **查看日志**
   - 保持终端窗口打开
   - 在浏览器中处理视频
   - 终端会显示详细的转录日志

**优点：**
- ✅ 使用现有脚本，无需额外配置
- ✅ 终端输出清晰

## 💡 提示

1. **性能监控**
   - 终端日志可以帮助您了解转录速度
   - 如果转录很慢，可以考虑使用更小的模型（tiny 或 base）

2. **调试问题**
   - 如果转录结果不理想，查看终端日志可以发现问题
   - 例如：语言识别错误、音频质量问题等

3. **保存日志**
   - 如果需要保存日志，可以使用重定向：
     ```bash
     python3 app_advanced.py 2>&1 | tee app.log
     ```

## 🔧 故障排除

### 问题 1：终端窗口没有打开
**解决方案：**
- 确保脚本有执行权限：`chmod +x "启动服务（带日志窗口）.command"`
- 手动在终端运行脚本

### 问题 2：看不到详细的转录日志
**解决方案：**
- 确认 `verbose=True` 已在 `video_transcriber.py` 中设置
- 检查终端窗口是否被最小化

### 问题 3：日志输出太快，看不清
**解决方案：**
- 使用终端的滚动功能
- 或者将日志保存到文件：`python3 app_advanced.py > output.log 2>&1`
