## 🚀 一键清理

### 运行清理脚本

```bash
./清理过时文件.sh
```

脚本会：
1. 列出所有将被删除的文件
2. 询问确认
3. 执行删除
4. 显示清理结果

### 手动清理

如果你想手动删除：

```bash
# 删除过时的安装脚本
rm 一键安装依赖.sh
rm 安装OCR依赖.sh

# 删除过时的启动脚本
rm 启动服务.sh
rm 启动客户端.command

# 删除重复的文档
rm QUICKSTART.md
rm 使用说明-增强版.md
rm 启动说明.md
rm 音频提取快速测试.md

# 删除过时的 Python 文件
rm ai_summarizer.py
rm ai_summarizer_v2.py
rm import_models.py

# 删除空目录
rmdir audio_output

# 清理系统文件
find . -name ".DS_Store" -delete
find . -type d -name "__pycache__" -exec rm -rf {} +
find . -name "*.pyc" -delete
```

## 🔄 如果需要恢复

如果删除后发现问题，可以从 Git 恢复：

```bash
# 查看删除的文件
git log --diff-filter=D --summary

# 恢复单个文件
git checkout HEAD~1 -- 文件名

# 恢复所有删除
git reset --hard HEAD~1
```

**建议**: 先运行清理脚本查看将删除的文件，确认无误后再执行。
