## 🚀 离线安装步骤

### 前提条件

1. **Python 3.9+** 已安装
2. **FFmpeg** 已安装（音频提取必需）
3. 已将整个项目文件夹复制到目标机器

### 步骤 1：创建虚拟环境

```bash
cd /path/to/video-to-text-tool
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate  # Windows
```

### 步骤 2：离线安装 Python 依赖

```bash
# 从本地 offline_packages 目录安装所有依赖
pip install --no-index --find-links=offline_packages -r requirements.txt
```

**参数说明**：
- `--no-index` - 不使用 PyPI 索引（纯离线）
- `--find-links=offline_packages` - 从本地目录查找包

### 步骤 3：安装 Whisper 模型

```bash
# 创建 Whisper 缓存目录
mkdir -p ~/.cache/whisper

# 复制模型文件
cp offline_packages/whisper_models/base.pt ~/.cache/whisper/
```

### 步骤 4：配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入您的 API Key
nano .env  # 或使用其他编辑器
```

**必需配置**：
```
OPENAI_API_KEY=your-api-key-here
OPENAI_BASE_URL=https://api.openai.com/v1
```

### 步骤 5：启动服务

```bash
# 方法 1：使用启动脚本（推荐）
./启动服务.sh

# 方法 2：手动启动
export KMP_DUPLICATE_LIB_OK=TRUE
source venv/bin/activate
python3 app_advanced.py
```

服务将在 **http://localhost:5000** 启动。

## 🔧 故障排除

### 问题 1：pip 安装失败

**错误**：`Could not find a version that satisfies the requirement...`

**解决**：
```bash
# 检查 offline_packages 目录是否完整
ls -lh offline_packages/ | wc -l
# 应该显示 44+ 个文件

# 确保使用正确的 Python 版本
python3 --version
# 应该是 3.9 或更高
```

### 问题 2：Whisper 模型未找到

**错误**：`Model 'base' not found`

**解决**：
```bash
# 检查模型文件是否存在
ls -lh ~/.cache/whisper/base.pt

# 如果不存在，重新复制
cp offline_packages/whisper_models/base.pt ~/.cache/whisper/
```

### 问题 3：FFmpeg 未安装

**错误**：`ffmpeg not found`

**解决**：

**macOS**：
```bash
brew install ffmpeg
```

**Ubuntu/Debian**：
```bash
sudo apt-get install ffmpeg
```

**Windows**：
- 下载 FFmpeg：https://ffmpeg.org/download.html
- 添加到系统 PATH

### 问题 4：OpenMP 库冲突

**错误**：`OMP: Error #15: Initializing libiomp5.dylib...`

**解决**：
```bash
# 启动脚本已自动设置，如果手动启动需要：
export KMP_DUPLICATE_LIB_OK=TRUE
python3 app_advanced.py
```

### 基本使用

1. **打开浏览器**：访问 http://localhost:5000
2. **上传视频**：拖拽或点击上传（支持 mp4, avi, mov 等）
3. **选择模型**：
   - Whisper 模型：base（推荐）
   - AI 模型：根据配置选择
4. **开始处理**：点击"开始处理"按钮
5. **查看结果**：转录完成后可下载文本或 AI 摘要

### 高级功能

- **批量处理**：一次上传多个视频
- **手动排序**：拖拽调整处理顺序
- **自定义 AI 提示词**：在 API 设置中配置
- **实时日志**：右侧窗口显示处理进度

## 📦 更新离线包

如果需要更新离线包（在有网络的机器上）：

```bash
# 1. 更新 requirements.txt
pip install -r requirements.txt --upgrade

# 2. 重新下载所有包
rm -rf offline_packages/*.whl
pip download -r requirements.txt -d offline_packages

# 3. 更新 Whisper 模型（如果需要其他模型）
# tiny 模型（最快）
python3 -c "import whisper; whisper.load_model('tiny')"
cp ~/.cache/whisper/tiny.pt offline_packages/whisper_models/

# small 模型（更准确）
python3 -c "import whisper; whisper.load_model('small')"
cp ~/.cache/whisper/small.pt offline_packages/whisper_models/
```

## 💡 性能优化建议

### 1. 选择合适的模型

- **快速处理**：使用 tiny 模型
- **日常使用**：使用 base 模型（推荐）
- **高质量**：使用 small 模型

### 2. 系统要求

**最低配置**：
- CPU: 2 核心
- 内存: 4 GB
- 存储: 2 GB 可用空间

**推荐配置**：
- CPU: 4 核心+
- 内存: 8 GB+
- 存储: 5 GB+ 可用空间

### 3. 处理时间参考

**10 分钟视频**（base 模型，CPU）：
- 2 核 CPU: 10-15 分钟
- 4 核 CPU: 5-8 分钟
- 8 核 CPU: 3-5 分钟
