# 🇨🇳 国内镜像源配置指南

为了加速下载，本项目已配置多个国内镜像源。

## 📦 pip 镜像源

### 自动配置（推荐）

运行环境检查脚本会自动配置：

```bash
./环境检查与安装.sh
```

### 手动配置

#### macOS / Linux

创建或编辑 `~/.pip/pip.conf`：

```bash
mkdir -p ~/.pip
cat > ~/.pip/pip.conf << 'EOF'
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn

[install]
trusted-host = pypi.tuna.tsinghua.edu.cn
EOF
```

#### Windows

创建或编辑 `%APPDATA%\pip\pip.ini`：

```ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn

[install]
trusted-host = pypi.tuna.tsinghua.edu.cn
```

## 🤗 HuggingFace 镜像源

Whisper 模型下载使用 HuggingFace，可配置镜像加速：

### 方式一：环境变量（临时）

```bash
# macOS/Linux
export HF_ENDPOINT=https://hf-mirror.com

# Windows (PowerShell)
$env:HF_ENDPOINT="https://hf-mirror.com"

# Windows (CMD)
set HF_ENDPOINT=https://hf-mirror.com
```

### 方式二：永久配置

**macOS/Linux** - 添加到 `~/.bashrc` 或 `~/.zshrc`：

```bash
echo 'export HF_ENDPOINT=https://hf-mirror.com' >> ~/.zshrc
source ~/.zshrc
```

**Windows** - 添加系统环境变量：

1. 右键"此电脑" → 属性 → 高级系统设置
2. 环境变量 → 新建用户变量
3. 变量名：`HF_ENDPOINT`
4. 变量值：`https://hf-mirror.com`

### HuggingFace 镜像源列表

| 镜像源 | URL | 说明 |
|--------|-----|------|
| HF-Mirror | https://hf-mirror.com | 推荐，速度快 |
| ModelScope | https://modelscope.cn | 阿里云，需注册 |

## 🎯 Whisper 模型下载

### 使用镜像下载

```bash
# 设置镜像
export HF_ENDPOINT=https://hf-mirror.com

# 下载模型
python -c "import whisper; whisper.load_model('base')"
```

### 模型大小参考

| 模型 | 大小 | 下载时间（参考） |
|------|------|------------------|
| tiny | ~75MB | 1-2 分钟 |
| base | ~150MB | 2-5 分钟 |
| small | ~500MB | 5-10 分钟 |
| medium | ~1.5GB | 15-30 分钟 |
| large | ~3GB | 30-60 分钟 |

### 手动下载模型（备选方案）

如果自动下载失败，可以手动下载：

1. 访问 https://hf-mirror.com/openai/whisper-base
2. 下载模型文件
3. 放到缓存目录：
   - macOS/Linux: `~/.cache/whisper/`
   - Windows: `%USERPROFILE%\.cache\whisper\`

## 🌐 Git 镜像加速

### GitHub 镜像

如果需要克隆 GitHub 仓库：

```bash
# 使用 GitHub 镜像
git clone https://ghproxy.com/https://github.com/user/repo.git

# 或使用 FastGit
git clone https://hub.fastgit.xyz/user/repo.git
```

### 配置 Git 代理（可选）

```bash
# HTTP 代理
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890

# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
```

## 🔧 Homebrew 镜像（macOS）

### 配置清华镜像

```bash
# 替换 Homebrew 源
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"

# 更新
brew update
```

### 永久配置

添加到 `~/.zshrc`：

```bash
cat >> ~/.zshrc << 'EOF'
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"
EOF

source ~/.zshrc
```

## ⚠️ 常见问题

### Q1: pip 镜像配置后仍然很慢？

**A:** 尝试其他镜像源，或检查网络连接：

```bash
# 测试网络
ping pypi.tuna.tsinghua.edu.cn

# 尝试其他镜像
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

### Q2: SSL 证书错误？

**A:** 添加 `trusted-host`：

```bash
pip install package_name -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn
```

### Q3: Whisper 模型下载超时？

**A:** 
1. 设置 HF_ENDPOINT 环境变量
2. 使用手动下载方式
3. 增加超时时间：
   ```bash
   export REQUESTS_TIMEOUT=300
   ```

### Q4: 镜像源不可用？

**A:** 镜像源可能维护中，切换到其他镜像：

```bash
# 切换到阿里云
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
```

## 🔄 恢复默认源

### pip 恢复默认

```bash
# macOS/Linux
rm ~/.pip/pip.conf

# Windows
del %APPDATA%\pip\pip.ini
```

### HuggingFace 恢复默认

```bash
# macOS/Linux
unset HF_ENDPOINT

# Windows (PowerShell)
Remove-Item Env:\HF_ENDPOINT
```
