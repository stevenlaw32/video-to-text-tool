### 📝 使用建议

#### 推荐工作流程：

1. **方式A：本地上传（最推荐）** ⭐⭐⭐⭐⭐
   ```
   浏览器扩展下载视频 → 上传到工具 → 自动转录 + AI摘要
   ```

2. **方式B：直接链接解析** ⭐⭐⭐⭐
   ```
   粘贴视频链接 → yt-dlp 自动下载 → 转录 + AI摘要
   ```

3. **方式C：音频处理** ⭐⭐⭐⭐
   ```
   上传视频 → 提取音频 → 下载使用
   ```

### 🔧 技术栈

- **Python**: 3.11.15
- **Web框架**: Flask 3.0.0
- **AI模型**: Whisper (OpenAI)
- **视频处理**: FFmpeg, OpenCV
- **下载工具**: yt-dlp
- **前端**: TailwindCSS, Vanilla JS

### 📊 性能指标

- **转录速度**: 取决于Whisper模型
  - tiny: 最快（实时的2-3倍）
  - base: 推荐（实时的1-2倍）
  - small: 更准确（接近实时）

- **支持格式**: 
  - 视频: mp4, avi, mov, mkv, flv, wmv
  - 音频: mp3, wav, m4a, flac
