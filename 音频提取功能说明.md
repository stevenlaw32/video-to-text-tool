# 🎵 音频提取功能说明

## 功能概述

从视频文件中提取音频并转换为多种音频格式（MP3、WAV、AAC、FLAC、OGG），支持批量处理和自定义音质设置。

## 快速开始

### 1. 访问音频提取页面

启动服务后，访问：`http://localhost:5000/audio`

或从主页点击右上角的 **🎵 音频提取** 按钮

### 2. 上传视频文件（支持批量）

- **拖拽上传**：将多个视频文件拖拽到上传区域
- **点击选择**：点击上传区域，可按住 Ctrl/Cmd 多选文件
- **批量处理**：一次可上传多个文件，自动批量提取
- **支持格式**：MP4, AVI, MOV, MKV, FLV, WMV, M4V, WEBM

### 3. 配置提取设置

- **输出格式**：MP3（默认）、WAV、AAC、FLAC、OGG
- **音质**：128k、192k（默认）、256k、320k
- **采样率**：44.1 kHz（默认）、48 kHz

### 4. 开始提取

点击 **开始提取** 按钮，系统会自动处理所有文件

### 5. 下载音频

- **单个下载**：点击文件列表中的下载按钮
- **批量下载**：点击 **批量下载** 按钮

### ✨ 核心功能

- ✅ **多格式支持**：支持5种常用音频格式输出

## API 接口

### 1. 提取音频

```http
POST /api/audio/extract
Content-Type: application/json

{
  "video_path": "/path/to/video.mp4",
  "format": "mp3",
  "bitrate": "192k",
  "sample_rate": 44100
}
```

**响应**：
```json
{
  "success": true,
  "output_path": "/path/to/output.mp3",
  "output_filename": "output.mp3",
  "file_size": 5242880,
  "format": "mp3",
  "bitrate": "192k",
  "sample_rate": 44100
}
```

### 2. 批量提取

```http
POST /api/audio/batch_extract
Content-Type: application/json

{
  "video_paths": ["/path/to/video1.mp4", "/path/to/video2.mp4"],
  "format": "mp3",
  "bitrate": "192k",
  "sample_rate": 44100
}
```

### 3. 下载音频

```http
GET /api/audio/download/<filename>
```

### 4. 获取音频信息

```http
GET /api/audio/info/<filename>
```

**响应**：
```json
{
  "success": true,
  "duration": 180.5,
  "bitrate": 192000,
  "sample_rate": 44100,
  "channels": 2,
  "codec": "mp3",
  "file_size": 5242880
}
```

### 5. 列出输出文件

```http
GET /api/audio/list
```

### 6. 删除文件

```http
DELETE /api/audio/delete/<filename>
```

## 音频格式说明

| 格式 | 编码器 | 特点 | 适用场景 |
|------|--------|------|----------|
| **MP3** | libmp3lame | 通用性最强 | 日常使用、分享 |
| **WAV** | pcm_s16le | 无损音质 | 专业编辑 |
| **AAC** | aac | 高压缩比 | 移动设备 |
| **FLAC** | flac | 无损压缩 | 音乐收藏 |
| **OGG** | libvorbis | 开源格式 | 游戏、网页 |

---

## 音质参数说明

### 比特率（Bitrate）

- **128 kbps**：标准质量，文件较小
- **192 kbps**：高质量（推荐），平衡音质和大小
- **256 kbps**：极高质量，接近CD音质
- **320 kbps**：最高质量，文件较大

### 采样率（Sample Rate）

- **44.1 kHz**：CD标准，适合大多数场景
- **48 kHz**：专业级，用于视频制作

## 常见问题

### Q1: 提取失败怎么办？

**A**: 检查以下几点：
1. 确认视频文件完整且未损坏
2. 确认FFmpeg已正确安装
3. 查看错误信息，可能是格式不支持
4. 尝试更换输出格式

### Q2: 提取的音频文件在哪里？

**A**: 音频文件保存在项目根目录的 `audio_output/` 文件夹中

### Q3: 可以提取视频中的某一段音频吗？

**A**: 当前版本提取完整音频，后续版本会支持时间段选择

### Q4: 支持哪些视频格式？

**A**: 支持所有FFmpeg支持的视频格式，包括：
- MP4, AVI, MOV, MKV
- FLV, WMV, M4V, WEBM
- 以及更多...

### Q5: 批量处理时可以中断吗？

**A**: 可以，刷新页面即可中断。已完成的文件会保留。

## 性能优化建议

1. **批量处理**：一次处理多个文件比分次处理更高效
2. **格式选择**：MP3格式处理速度最快
3. **音质平衡**：192k是音质和大小的最佳平衡点
4. **定期清理**：及时删除不需要的输出文件
