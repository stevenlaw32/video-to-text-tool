## 🚀 快速使用

### **方法 1：自动更新所有Cookie（推荐）**

```bash
source venv/bin/activate
python cookie_manager.py --update
```

这会自动从你的浏览器（Chrome/Safari/Edge/Firefox）获取抖音和小红书的Cookie。

### **方法 2：只更新抖音Cookie**

```bash
python cookie_manager.py --douyin
```

### **方法 3：只更新小红书Cookie**

```bash
python cookie_manager.py --xiaohongshu
```

### **方法 4：查看Cookie状态**

```bash
python cookie_manager.py --status
```

## 📋 使用前提

### **1. 在浏览器中登录**

- 打开浏览器，访问 `https://www.douyin.com`

- Safari 需要关闭浏览器才能读取Cookie

### **3. 运行自动更新命令**

```bash
python cookie_manager.py --update
```

## ⚠️ 重要提示

### **关于抖音的反爬虫**

抖音有严格的反爬虫机制，即使有Cookie也可能被检测。如果遇到解析失败：

1. **使用分享链接**
   - 在抖音APP中点击"分享"
   - 复制分享链接（如 `https://v.douyin.com/xxxxx/`）
   - 这种短链接更容易解析

2. **定期更新Cookie**
   - Cookie会过期（通常7-30天）
   - 定期运行 `python cookie_manager.py --update`

3. **避免频繁请求**
   - 不要短时间内解析大量视频
   - 建议每次请求间隔3-5秒

### **支持的浏览器**

✅ Chrome - 完美支持  
✅ Edge - 完美支持  
✅ Firefox - 支持  
⚠️ Safari - 需要关闭浏览器

## 🛠️ 故障排除

### **问题1：提示"未能获取Cookie"**

**解决方案：**
1. 确保已在浏览器中登录
2. 尝试关闭浏览器后再运行
3. 检查浏览器是否支持

### **问题2：解析仍然失败**

**解决方案：**
1. 检查Cookie是否过期：`python cookie_manager.py --status`
2. 重新登录网站并更新Cookie
3. 尝试使用分享链接而不是直接链接

### **问题3：Safari无法读取Cookie**

**解决方案：**
```bash
# macOS需要授予终端完全磁盘访问权限
# 系统偏好设置 → 安全性与隐私 → 完全磁盘访问权限
# 添加"终端"应用
```

## 💡 高级用法

### **在Python代码中使用**

```python

manager.show_cookie_status()
```
