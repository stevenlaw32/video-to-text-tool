# 🚀 快速开始指南

## 📦 一键安装

### macOS / Linux

```bash
# 1. 进入项目目录
cd /path/to/video-to-text-tool

# 2. 运行环境检查脚本
./环境检查与安装.sh
```

### Windows

```batch
# 1. 进入项目目录
cd C:\path\to\video-to-text-tool

# 2. 双击运行或在命令行执行
环境检查与安装.bat
```

## ✅ 脚本会自动完成

- ✓ 检查 Python 版本（需要 3.8+）
- ✓ 检查 FFmpeg 是否安装
- ✓ 配置 pip 国内镜像源（清华大学镜像）
- ✓ 创建 Python 虚拟环境
- ✓ 安装所有 Python 依赖包
  - **优先使用 `offline_packages/` 离线包**（如果存在）
  - 离线包不完整时自动从网络补充
- ✓ 下载 Whisper 模型（base 模型）
  - **优先从 `offline_packages/whisper_models/` 复制**（如果存在）
  - 离线模型不存在时自动从网络下载
- ✓ 创建必要的配置文件
- ✓ 创建 output 和 uploads 目录

## 📦 离线安装支持

如果项目包含 `offline_packages/` 目录，脚本会**优先使用离线包**安装，无需网络连接！

详见：[离线安装指南.md](离线安装指南.md)

## 🔧 手动安装（如果脚本失败）

### 1. 安装系统依赖

**macOS:**
```bash
brew install ffmpeg
```

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install ffmpeg
```

**Windows:**
- 下载 FFmpeg: https://ffmpeg.org/download.html
- 解压并添加到系统 PATH

### 2. 配置 pip 镜像源

**macOS/Linux:**
```bash
mkdir -p ~/.pip
cat > ~/.pip/pip.conf << EOF
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
EOF
```

**Windows:**
```batch
mkdir %APPDATA%\pip
echo [global] > %APPDATA%\pip\pip.ini
echo index-url = https://pypi.tuna.tsinghua.edu.cn/simple >> %APPDATA%\pip\pip.ini
echo trusted-host = pypi.tuna.tsinghua.edu.cn >> %APPDATA%\pip\pip.ini
```

### 3. 创建虚拟环境并安装依赖

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
# macOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 升级 pip
pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

### 4. 配置 API 密钥

```bash
# 复制配置文件
cp models.json.example models.json
cp ocr_apis.json.example ocr_apis.json
cp .env.example .env

# 编辑配置文件，填入你的 API 密钥
# macOS/Linux:
nano models.json
# Windows:
notepad models.json
```

## 🔍 常见问题

### Q1: pip 安装速度慢？
**A:** 脚本已自动配置清华镜像源，如果还是慢可以尝试：
```bash
# 阿里云镜像
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/

# 中科大镜像
pip install -r requirements.txt -i https://pypi.mirrors.ustc.edu.cn/simple/
```

### Q2: Whisper 模型下载失败？
**A:** 模型会在首次使用时自动下载，如果失败：
```bash
# 设置 HuggingFace 镜像
export HF_ENDPOINT=https://hf-mirror.com

# 手动下载
python -c "import whisper; whisper.load_model('base')"
```

### Q3: FFmpeg 未找到？
**A:** 
- macOS: `brew install ffmpeg`
- Ubuntu: `sudo apt-get install ffmpeg`
- Windows: 下载后添加到 PATH

### Q4: 虚拟环境激活失败？
**A:** 
```bash
# macOS/Linux 权限问题
chmod +x venv/bin/activate

# Windows 执行策略问题
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Q5: 端口 5000 被占用？
**A:** 修改 `app_advanced.py` 中的端口：
```python
app.run(host='0.0.0.0', port=5001, debug=False)
```
