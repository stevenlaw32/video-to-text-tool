## 🔧 核心 Python 文件

### 主应用

| 文件 | 说明 |
|------|------|
| `app_advanced.py` | 主 Web 应用（Flask），包含所有路由 |
| `main.py` | 命令行工具入口 |

### 功能模块

| 文件 | 说明 |
|------|------|
| `video_transcriber.py` | 视频转文字核心模块（Whisper） |
| `video_ocr.py` | 视频 OCR 文字识别模块 |
| `audio_extractor.py` | 音频提取模块 |
| `api_client.py` | AI API 客户端（GPT/Claude） |
| `cloud_ocr.py` | 云端 OCR API（百度/腾讯/阿里） |
| `simple_ocr.py` | 本地 OCR（Surya） |

### 配置管理

| 文件 | 说明 |
|------|------|
| `api_config.py` | API 配置管理类 |
| `models_config.py` | 多模型配置管理类 |
| `ocr_config.py` | OCR 配置管理类 |

### 路由模块

| 文件 | 说明 |
|------|------|
| `models_routes.py` | 模型管理相关路由 |
| `api_routes.py` | API 配置相关路由 |
| `ocr_routes.py` | OCR 功能相关路由 |
| `audio_routes.py` | 音频提取相关路由 |

### 工具模块

| 文件 | 说明 |
|------|------|
| `log_stream.py` | 实时日志流系统 |
| `import_models.py` | 模型导入工具 |

## 💡 提示

1. **首次使用**: 运行 `环境检查与安装.sh` 自动配置
2. **配置 API**: 编辑 `models.json` 添加你的 API 密钥
3. **启动服务**: 运行 `启动服务（带日志窗口）.command`
4. **查看日志**: 访问 `logs/app.log`
5. **输出文件**: 在 `output/` 目录查看结果
