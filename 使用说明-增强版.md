## ✨ 核心功能

### 1️⃣ 批量拖拽上传
- **多文件上传**：一次性拖拽或选择多个视频文件
- **支持格式**：MP4, AVI, MOV, MKV, FLV, WMV, M4V, WEBM
- **实时预览**：上传后立即显示文件列表

### 2️⃣ 手动调整顺序
- **拖拽排序**：点击视频左侧的拖拽图标，上下拖动调整顺序
- **序号显示**：每个视频前显示处理序号
- **实时更新**：拖动后序号自动更新

### 3️⃣ 自动衔接结果
- **按序处理**：严格按照你调整的顺序处理视频
- **自动合并**：所有转录文本按顺序自动衔接
- **章节标记**：每个视频的内容会自动添加标题分隔

### 4️⃣ 自定义AI模型
- **Whisper模型选择**：
  - Tiny - 最快，适合快速预览
  - Base - 推荐，平衡速度和准确度
  - Small - 较准确，稍慢
  - Medium - 很准确，较慢
  - Large - 最准确，最慢

- **AI摘要模型**：
  - 默认使用 gpt-4o
  - 可自定义任何OpenAI兼容模型
  - 支持修改API设置

### 5️⃣ 自定义提示词
- **编辑提示词**：点击"编辑"按钮打开提示词编辑器
- **占位符**：使用 `{transcript}` 作为转录内容的占位符
- **保存模板**：自定义的提示词会保存在当前会话中
- **恢复默认**：随时可以恢复使用默认提示词

## 📝 使用流程

### 步骤1：上传视频
1. 点击上传区域或直接拖拽视频文件
2. 支持一次上传多个文件
3. 上传后会显示在视频列表中

### 步骤2：调整顺序
1. 在视频列表中，点住左侧的拖拽图标
2. 上下拖动调整视频处理顺序
3. 序号会自动更新

### 步骤3：配置设置
1. **选择Whisper模型**：根据需求选择识别模型
2. **选择语言**：中文、英文、日文、韩文等
3. **启用AI摘要**：勾选后会生成AI整理的摘要
4. **设置AI模型**：可以修改使用的AI模型名称
5. **自定义提示词**（可选）：点击"编辑"自定义AI整理的方式

### 步骤4：开始处理
1. 点击"开始处理"按钮
2. 等待进度条完成
3. 查看结果

### 步骤5：查看和下载
1. **完整转录文本**：所有视频按顺序合并的文本
2. **AI摘要**：AI整理后的结构化文档（如果启用）
3. 点击"下载"按钮保存结果

## 💡 高级技巧

### 技巧1：批量处理不同类型视频
如果有多种类型的视频，可以分批处理：
1. 先上传同类型的视频
2. 处理完成后下载结果
3. 清空列表
4. 上传下一批视频

### 技巧2：自定义提示词模板
常用提示词模板：

**教程格式：**
```
请将以下视频转录整理成教程文档，要求：
1. 提取主题和核心内容
2. 按逻辑顺序组织成章节
3. 列出关键要点和步骤
4. 使用Markdown格式

转录内容：{transcript}
```

**摘要格式：**
```
请对以下视频内容进行总结：
1. 概括主要内容（3-5句话）
2. 列出关键要点（5-10条）
3. 提取重要结论

转录内容：{transcript}
```

**笔记格式：**
```
请整理成学习笔记：
1. 清晰的标题和子标题
2. 核心概念和定义
3. 重要示例
4. 注意事项

转录内容：{transcript}
```

### 技巧3：优化处理速度
- 使用 Tiny 或 Base 模型
- 如果只需要文字不需要AI整理，取消勾选"启用AI摘要"
- 视频数量较多时，可以分批处理

### 技巧4：提高识别准确度
- 使用 Small 或 Medium 模型
- 确保视频音质清晰
- 选择正确的识别语言

## 🔧 常见问题

### Q: 视频上传后看不到？
A: 检查视频格式是否支持，刷新页面重试

### Q: 处理速度很慢？
A: 
- 使用更小的Whisper模型（Tiny或Base）
- 减少单次处理的视频数量
- 检查电脑性能和内存

### Q: AI摘要生成失败？
A: 
- 检查API密钥是否正确
- 确认API额度是否充足
- 检查网络连接
- 查看控制台错误信息

### Q: 如何保存自定义提示词？
A: 当前版本提示词保存在会话中，刷新页面后需要重新设置。可以将常用提示词保存到本地文本文件中。

### Q: 可以同时处理多少个视频？
A: 理论上没有限制，但建议单次处理不超过20个视频，以确保稳定性。
